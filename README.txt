ENROL PLUGIN Custom redirect
============================
A moodle enrol plugin to redirect a user to custom URI when is not enroled in course.

Package tested in: moodle 3.7+

QUICK INSTALL
==============
Download zip package, extract the customredirect folder and upload this folder into enrol/.
